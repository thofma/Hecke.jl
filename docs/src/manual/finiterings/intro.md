# Introduction

A finite ring will is represented as an abelian group $A$ together with a $\mathbf{Z}$-bilinear map $A \times A \to A$ encoding the multiplicative structure.

The type for finite rings is `FiniteRing` and the type for the elements is `FiniteRingElem.`
