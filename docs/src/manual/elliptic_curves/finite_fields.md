## Random points

```
  rand(E::EllipticCurve{<: FinFieldElem})
```

Return a random point on the elliptic curve $E$ defined over a finite field.

```jldoctest; filter = r"\(.*"
julia> E = elliptic_curve(GF(3), [1, 2]);

julia> rand(E)
(2 : 0 : 1)
```
